# MonoReg PRD

## 产品定位

MonoReg 是面向竞赛、训练营、考试、讲座和大型活动的轻量报名管理系统。第一版聚焦管理员创建活动、配置动态报名表、公开报名、数据管理、账号发放和邮件通知闭环。

## 用户角色

- `super_admin`: 全部权限。
- `admin`: 活动、报名、账号、邮件日常管理权限。
- `viewer`: 预留只读角色，后端已限制写操作。

## 核心流程

1. 管理员登录后台。
2. 创建活动并设置报名时间、人数上限、账号和邮件策略。
3. 配置动态表单字段。
4. 用户访问 `/register/:slug` 提交报名。
5. 管理员查看报名、筛选、导出、批量审批。
6. 管理员生成或导入账号。
7. 管理员创建邮件模板，预览变量，批量发送确认或账号邮件。
8. Worker 异步发送邮件并写入日志。

## 验收点

- `docker compose up --build` 可启动 PostgreSQL、Redis、backend、worker、frontend。
- 默认管理员可初始化并登录。
- 活动、动态表单、公开报名、报名列表形成最小闭环。
- 账号密码加密保存，日志不保留明文密码。
- 批量邮件不阻塞 HTTP 请求。
- UI 保持黑白灰 Monochrome 风格。
